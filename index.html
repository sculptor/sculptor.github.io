The complete solution

Easy to learn, intuitive syntax of the textual DSL, based on the concepts from DDD.

Free and open source

Released under the Apache 2.0 license, source available on GitHub.

Rock-solid foundation

The generated code is based on well-known frameworks, such as JPA, Hibernate, Spring Framework, Spring Web Flow, and Java EE.
